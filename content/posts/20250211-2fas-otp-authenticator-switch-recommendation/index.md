**2FA Authenticator (2FAS)**

2fas.com

## 개인 코멘트
2FA OTP 앱을 갈아탔다.
앞으로 쭉 정착할 수 있을 것 같아 추천.

### 채택 조건
- 폰, 데스크톱, 워치 모두 지원할 것
- 클라우드 기반 백업 및 동기화 가능할 것
- 앱 만듦새가 적당히 문제없을 것
- 보안 우려가 없고 운영 주체를 신뢰할 수 있을 것
- 다른 서비스에서 넘어올 때를 위해 데이터 불러오기가 가능할 것
- 다른 서비스로 갈아탈 때를 위해 데이터 내보내기가 가능할 것

### 그간 써본 것과 더 이상 쓰지 않는 이유
- **구글 인증기**, 데스크톱과 워치 지원 안 함
- **트윌리오 오씨**, 데스크톱 지원 안 함, 데이터 유출 이력 있음
- **조호 원오스**, 이 회사에서 이 서비스가 주 상품이 아님
